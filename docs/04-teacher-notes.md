# 선생님 노하우 — 게임 개발 & 클로드코드 실전 영역

> 학생 버전과 선생님 버전의 차이는 기능이 아니라 **습관**이다. 이 문서가 그 습관의 목록이다.

## 1. 게임 코드의 노하우

### 상태머신을 명시적으로
`state = 'title' | 'lobby' | 'play' | 'cleared' | 'over'` — 까비는 이미 이 패턴을 쓰고 있다 (잘함).
노하우: 상태 전이를 한 함수(`setState(next)`)로 모으면 "일시정지 중 죽음" 같은 버그가 사라진다.
전이 함수 안에서만 overlay 표시/음악/입력 초기화를 처리한다.

### 매직 넘버는 config로
`e.speed = 0.9 + Math.random()*0.5 + difficulty*0.12` 같은 수치가 코드 곳곳에 흩어지면 밸런스 조정이 지옥이 된다.
모든 수치를 `config.js` 한 곳에 모으고 이름을 붙인다 (`ENEMY_BASE_SPEED`). **밸런스 조정 = config 파일 수정**이 되게.

### 프레임 의존을 끊는다 (deltaTime)
지금 코드는 "1프레임 = 1틱". 빠른 모니터(120Hz)에서는 게임이 2배 빨라진다.
`requestAnimationFrame`의 타임스탬프로 `dt`를 구해 `x += speed * dt`로 바꾸는 것이 진짜 게임의 첫걸음.
(M2에서 함께 적용 — 한 번에 다 바꾸면 손맛이 변하니 수치 재조정 필수)

### 저장은 스키마부터
```js
const SAVE_KEY = 'kkabbi-save';
const SAVE_VERSION = 1;
// { v: 1, best: 0, cleared: ["dungeon1"], lastChar: "mareh" }
```
- 읽을 때 `v`가 다르면 버리고 기본값 — 옛 데이터로 게임이 깨지는 것보다 백배 낫다
- try/catch 필수 — 시크릿 모드에서 localStorage는 던진다(throw)

### 게임필(juice)은 싸고 효과가 크다
히트스톱(맞는 순간 30ms 멈춤), 화면 흔들림 4px, 피격 플래시 — 합쳐서 30줄이면
"허접한 게임"이 "손맛 있는 게임"이 된다. 그래픽보다 먼저 할 것.

### 디버그 모드를 만들어 둔다
`?debug=1`이면: 무적, 히트박스 표시, 스테이지 점프 키. 테스트 시간이 1/10이 된다.

## 2. 클로드코드 사용 노하우 (이 저장소의 작업 규칙)

### 문서 → 계획 → 작은 변경 → 확인
1. 기능 시작 전 `docs/01-PRD.md`와 대조 — 범위 밖이면 안 만든다
2. 큰 변경은 **Plan Mode**(Shift+Tab)로 계획부터 검토
3. 한 번에 한 기능. "저장 기능 만들고 모바일도 하고 사운드도" ❌
4. 변경 후 즉시 브라우저 실행 확인 — 확인 전엔 다음 단계 금지 (CLAUDE.md에도 명시)

### 프롬프트 노하우
- ❌ "게임 더 재밌게 해줘" → ✅ "적 처치 시 30ms 히트스톱과 4px 화면 흔들림을 추가해줘"
- 수치를 같이 준다: "점프가 무거워. jumpV를 -9.0에서 -9.6으로 바꿔보고 비교하게 해줘"
- 망가지면 고치라고 하기 전에 **/rewind** — 대화를 되감는 게 디버깅보다 빠를 때가 많다
- 코드를 읽게 한다: "이 충돌 판정 코드를 초보자에게 설명해줘" — 이해 없이 누적된 코드가 기술 부채

### 커밋 노하우
- 커밋 = 동작하는 최소 단위. "저장 기능 추가" 하나, "밸런스 조정" 하나
- 메시지는 "무엇을 왜": `저장: 최고 기록 localStorage 저장 (M1) — 새로고침해도 유지되게`
- 동작 안 하는 코드는 커밋하지 않는다

## 3. 흔한 함정 (학생이 빠지기 쉬운 곳)

| 함정 | 처방 |
|------|------|
| 기능을 계속 추가하고 완성을 안 함 | PRD의 "하지 않을 것" 섹션을 같이 읽는다 |
| AI가 준 코드를 이해 없이 누적 | 머지 전 "이 코드 설명해줘" 1회 의무화 |
| 한 번에 큰 리팩토링 | 보이스카우트 규칙 — 건드리는 부분만 정리 |
| 밸런스를 감으로 | 수치를 config로 모으고, 바꾼 값과 느낌을 기록 |
| 브라우저 캐시에 속음 | 강력 새로고침(Cmd+Shift+R)을 습관으로 |
